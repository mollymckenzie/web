# Datasheet Requirements (Datasheets for Datasets)

Inspired by "Datasheets for Datasets" (Gebru et al.), capture the following for each dataset:

- Motivation: Why was the dataset created?
- Composition: What data instances, labels, modalities?
- Collection Process: How was data collected and by whom?
- Preprocessing/Cleaning: What transformations were applied?
- Uses: Intended tasks and known limitations.
- Distribution: Licensing, access, citation.
- Maintenance: Updates, versioning, contact.

See template at `docs/templates/datasheet-template.md`.
