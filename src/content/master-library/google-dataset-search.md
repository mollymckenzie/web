Google Dataset Search is a specialized search engine that helps users discover datasets across thousands of repositories around the web.

## How It Works

Unlike general web search, Dataset Search indexes datasets that have been marked up with structured schema.org metadata. It allows you to filter by:
- License type (open, commercial use, etc.)
- Last updated
- Format (CSV, JSON, API, etc.)
- Topic area

## Use Cases

- **Finding data for research projects**: Search across government portals, academic repositories, and domain-specific data providers at once.
- **Teaching source evaluation**: Have students compare similar datasets from different sources and assess data quality, documentation, and licensing.
- **Discovering niche datasets**: Surface datasets that don't appear in standard Google search results.

## Tips for Effective Searching

- Use specific domain terms (e.g., "childhood obesity county level" rather than "obesity data")
- Filter by "Download" to find directly downloadable files
- Check the license before using any dataset for coursework or publication
