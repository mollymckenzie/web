The Knox County Property Assessor's Office maintains parcel-level records for all real property in Knox County, Tennessee.

## What's Included

- Parcel ID, address, and owner name
- Assessed value (land + improvements) and tax classification
- Land use code (residential, commercial, agricultural, etc.)
- Building characteristics: square footage, year built, bedrooms/baths
- Sale date and price history

## Accessing the Data

Data can be explored via the online parcel search tool or requested as a bulk extract. The GIS office publishes shapefiles and CSV exports of the full parcel database.

## Community Use

- **Housing affordability**: Track assessed value trends by neighborhood to analyze displacement risk.
- **Land use patterns**: Identify vacant lots, agricultural land, and commercial corridors.
- **Civic research**: Cross-reference ownership records with public records on tax delinquency.

## Classroom Use

- Map property values across ZIP codes using QGIS or ArcGIS Online.
- Compare land use distributions between urban and rural parts of Knox County.
- Discuss the relationship between assessed value and local school funding.
