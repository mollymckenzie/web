From Data to Viz is a decision tree that guides you toward the most appropriate chart type for your specific data structure and communication goal.

## What It Covers

- **Numeric data**: Distributions, correlations, rankings
- **Categorical data**: Part-of-whole, comparison, composition
- **Time series**: Trends, evolution, forecasting contexts
- **Maps**: Geographic data, spatial patterns
- **Network data**: Relationships, flows, hierarchies

## Key Features

Each chart type entry includes:
- A clear description of when to use it
- Multiple worked examples with real datasets
- Code in both R (ggplot2) and Python (matplotlib/seaborn)
- Common caveats and when *not* to use that chart type

## Teaching Suggestions

- Use the decision tree as a starting point for any visualization assignment
- Challenge students to find counter-examples where a recommended chart fails
- Compare the "caveats" sections to develop critical chart-reading skills
