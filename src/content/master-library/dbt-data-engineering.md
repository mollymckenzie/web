dbt (data build tool) has become the industry standard for SQL-based data transformation in modern data stacks. dbt Learn provides structured, free courses for learning data engineering fundamentals.

## Courses Available

- **dbt Fundamentals**: Core concepts — models, sources, tests, documentation, deployment
- **Jinja, Macros, Packages**: Advanced templating and code reuse in dbt projects
- **Advanced Materializations**: Incremental models, snapshots, and performance optimization
- **Analyses, Seeds, and Exposures**: Managing ad hoc analysis and dataset dependencies

## Prerequisites

- Solid SQL skills
- Familiarity with data warehouses (Snowflake, BigQuery, Redshift, or DuckDB)
- Basic command-line proficiency

## Why Teach This

dbt skills are in high demand in industry. Students learning dbt gain:
- Production-grade software engineering practices (version control, testing, documentation)
- Understanding of the ELT (Extract-Load-Transform) paradigm that dominates modern data infrastructure
- Portfolio-worthy projects using open datasets

## Note for Teachers

This resource is marked **teacher-only** because the advanced SQL and data warehouse requirements make it unsuitable for general student use. It is well-suited for data engineering capstone projects or graduate-level courses.
